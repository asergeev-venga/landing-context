# Golden Tasks (v1)

Use these tasks to benchmark generation quality and track regressions.

## Task GT-01: Hero section

Build a responsive hero section with:

- title
- subtitle
- primary CTA
- optional background asset

Expected files:

- `Section.tsx`
- `Section.module.scss`
- `index.ts`

## Task GT-02: Feature cards block

Build a 3-card feature section with:

- section heading
- card title + description
- responsive stack behavior on mobile

Expected files:

- `FeatureCards.tsx`
- `FeatureCards.module.scss`
- `index.ts`
